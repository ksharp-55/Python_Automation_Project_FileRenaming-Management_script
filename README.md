# Python_Automation_Project_FileRenaming-Management_script
Python automation script for file renaming and management to streamline workflows and improve efficiency


## Overview
This project automates file renaming and management to streamline workflows and reduce manual effort


## Key Objectives
-Automatically rename files in bulk
-Organize files into folders based on rules
-Reduce manual file management errors
-Improve efficiency and productivity


## Tools and Technologies
-Python for scripting
-Os and shutil libraries for file operations
-Regular expressions (regex) for pattern matching
-pandas for metadata management


## Description
This Python project automates common file management tasks such as renaming files base on rules, moving files into folders, and ensuring consistency in file organization. It demonstrates the use of Python script to handle repetitive tasks efficiently.


## Key Features
-Bulk file renaming based on naming conventions
-Automatic folder creation and file sorting
-Logging of file operations for verification
-Customizable rules for different use cases


## How to Use
1. Place files in the source directory
2. Configure renaming and sorting rules in the script
3. Run the Python automation script
4. Review output and logs for verification
